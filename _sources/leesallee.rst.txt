Índice Lee-Sallee
#############################

Para capas vectoriales

.. math::
    lee\_sallee\_index =  \frac{A\cap B}{A\cup B}


descarga el código de ejemplo 

:download:`indice_lee_sallee.py <../../codigos/indice_lee_sallee.py>`.

Insumos 
========

Descarga los insumos para este ejemplo :download:`aqui <../../lee_sallee/insumos/insumos_leesallee.zip>`

Procedimiento
==============
Abre el código **indice_lee_sallee.py** en Qgis 3.4 o superior, 
Si tienes dudas de como hacerlo visualiza la guia_

.. _guia: https://vichdzgeo.github.io/geo_lancis/ejecucion.html

Modifica las rutas donde se encuentran los insumos

* vlayer_base es la capa vectorial que se ocupa como base 

* vlayer_model es la capa vectorial que se ocupa como modelo
